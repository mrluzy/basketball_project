一、问题背景与任务说明
在自动控制与嵌入式智能系统的发展过程中，人工智能技术，尤其是神经网络模
型，正被广泛应用于解决传统控制问题。本课程设计以“机器人篮球竞赛中的投篮
控制器”为背景，要求学生设计一个能够根据输入信息预测最优投篮参数（初速
度、仰角、偏向角）的控制模型，并通过人工智能方法实现。
在本项目中，假设机器人位于一个已知平面场地上，其底盘可以自由运动，并携带
一个投篮装置。投篮装置支持俯仰角调整和偏向角控制。任务目标是：根据机器人
的当前位置 (x, y)、篮筐的位置 (x₀, y₀)、篮筐的高度 h，实时计算出最优的投篮初速
度、仰角和偏向角，使篮球能够准确投入篮筐。
本项目的核心挑战在于：
• 对机器人-场地-目标的空间关系进行物理建模；
• 在考虑扰动、误差等因素的情况下，获取充足的仿真或实验数据；
• 采用神经网络模型替代传统控制策略，完成控制器的设计；
• 实现简洁、可部署于嵌入式平台的神经网络结构，具备良好的实时性与推理
效率。
二、设计目标与要求
2.1. 系统建模与数据获取
• 建立简化的投篮物理模型（考虑空气阻力与不考虑均可，自主说明假设条
件）；
• 明确系统输入与输出变量（输入：[x, y, x₀, y₀, h]；输出：[v₀, θ_pitch, 
θ_yaw]）；
• 使用仿真工具（如 MATLAB / Python / Unity / 物理引擎等）生成数据集，必
须考虑适当的噪声干扰，例如传感器噪声、执行器误差、延迟等；
• 对生成数据进行预处理，划分训练集、验证集与测试集。
2.2. 人工智能模型设计
• 选用合适的神经网络模型结构（如前馈神经网络、轻量卷积网络、小型
Transformer 等），强烈推荐适用于实时实现与轻量化模型；
• 明确每一层的设计依据（激活函数、层数、神经元数目等），并进行理论分
析；
• 说明模型简化策略（如剪枝、量化、蒸馏、小模型设计等）以提升模型的嵌
入式适配性；
• 使用训练数据进行网络训练与调优，并记录损失下降与泛化能力。
2.3. 性能评估与测试
• 对训练后的模型在不同输入范围内进行测试，评估其误差分布；
• 分析模型输出与理想物理控制解之间的差异；
• 指出模型可能存在的失败案例或边界条件，并进行讨论；
• 探讨模型的嵌入式实现可行性，包括模型大小、计算复杂度、推理延迟等。
三、成果提交内容
（一）每位学生需提交一份完整的设计报告，包括但不限于以下内容：
3.1 问题建模与数据生成
• 控制问题的物理建模与假设条件
• 数据生成方法与工具
• 数据样本示意与分布分析
3.2 神经网络设计与实现
• 网络结构设计（包含结构图与参数表）
• 模型选择理由与技术路径分析
• 简化与优化措施（如剪枝、量化等）
3.3 训练过程与结果
• 损失函数选择
• 训练曲线展示与结果对比
• 超参数设置与调整过程说明
3.4 测试与性能评估
• 多种输入场景下的测试误差分布
• 与传统物理解的比较分析
• 失败情况剖析与鲁棒性探讨
3.5 嵌入式部署分析（可选加分项）
• 模型在树莓派、Jetson Nano 或 ESP32 等平台上的适配性探讨
• 推理时间评估与资源消耗分析
3.6 结论与未来改进方向

（二）学生需要提交课程设计源代码包。
